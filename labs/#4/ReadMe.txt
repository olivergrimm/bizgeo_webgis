Der Browser lässt den Zugriff auf das KML-File aus Sicherheitsgründen nicht zu (Cross origin requests).

Deshalb müssen wir unser HTML-File auf einem Web-Server laufen lassen. Einen einfachen Web-Server bietet folgende Seite an:
https://code.google.com/p/mongoose/

Das File mongoose-free-5.6.exe im selben Verzeichnis ablegen wie das HTML-File. Das Exe-File ausführen. Der Browser öffnet sich.  

Unser HTML-File muss aus dem Browser "Index of" aufgerufen werden.

Unter Windows kann mongoose unten rechts bei den ausgeblendeten Symbolen erneut angezeigt werden.
